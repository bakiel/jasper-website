## Nano Banana Pro Prompting Guide

### The ICS Framework

```
I = IMAGE TYPE
    Specify: infographic, diagram, flowchart, dashboard, etc.

C = CONTENT
    What information to display

S = STYLE
    Visual style: McKinsey, minimal, corporate, etc.
```

### Prompt Structure

```
[Subject + Adjectives] doing [Action] in [Location/Context].
[Composition/Camera Angle].
[Lighting/Atmosphere].
[Style/Media].
```

## Prompt Tips for JASPER

### Do's

```
✓ Always specify exact hex colours (#1E3A5F)
✓ Include precise dimensions
✓ Describe layout structure clearly
✓ Use "McKinsey" or "consulting" style references
✓ Specify "no stock photo aesthetic"
✓ Request "clean, minimal, professional"
✓ Describe spatial relationships precisely
```

### Don'ts

```
✗ Don't use vague colour descriptions
✗ Don't skip dimensions
✗ Don't request "cool" or "modern" without specifics
✗ Don't ask for actual DFI logos (trademark issues)
✗ Don't overcomplicate with too many elements
✗ Don't forget brand consistency references
```

### Testing Prompts

```
Before generating final assets, test with:

1. Single layer of complexity first
2. Verify colour accuracy
3. Check text rendering quality
4. Confirm layout matches intent
5. Iterate with refinements
```
