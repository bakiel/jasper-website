## Logo

### Primary Logo
The JASPER logo consists of:
1. **Logomark** - Crystalline hexagonal structure representing architectural precision
2. **Wordmark** - "JASPER" in bold, clean typography
3. **Tagline** - "FINANCIAL ARCHITECTURE" in small caps

### Logo Variants

| Variant | Use Case | File |
|---------|----------|------|
| **Full Horizontal** | Headers, presentations | `logo-full-horizontal.svg` |
| **Logomark Only** | Favicons, small spaces, social icons | `logo-mark.svg` |
| **Wordmark Only** | Text-heavy contexts | `logo-wordmark.svg` |
| **White Version** | Dark backgrounds | `logo-white.svg` |
| **Dark Version** | Light backgrounds | `logo-dark.svg` |

### Logo Clear Space
Minimum clear space around logo = height of the "J" in JASPER

### Minimum Size
- Horizontal logo: 120px width minimum
- Logomark only: 32px minimum

### Logo Don'ts
- ❌ Don't stretch or distort
- ❌ Don't change colours outside brand palette
- ❌ Don't add effects (shadows, gradients, outlines)
- ❌ Don't rotate
- ❌ Don't place on busy backgrounds without contrast

## Typography

### Primary Typeface: Poppins

**Poppins** is a geometric sans-serif with clean, modern aesthetics. It conveys professionalism and precision.

```css
font-family: 'Poppins', sans-serif;
```

### Font Weights

| Weight | Name | Usage |
|--------|------|-------|
| 300 | Light | Captions, fine print |
| 400 | Regular | Body text |
| 500 | Medium | Subheadings, emphasis |
| 600 | SemiBold | Section headers |
| 700 | Bold | Headlines, CTAs |

### Type Scale

| Element | Size (Desktop) | Size (Mobile) | Weight | Line Height |
|---------|---------------|---------------|--------|-------------|
| H1 | 48-60px | 32-40px | Bold | 1.1 |
| H2 | 36-48px | 28-32px | Bold | 1.2 |
| H3 | 24-30px | 20-24px | SemiBold | 1.3 |
| H4 | 18-20px | 16-18px | SemiBold | 1.4 |
| Body | 16px | 16px | Regular | 1.6 |
| Small | 14px | 14px | Regular | 1.5 |
| Caption | 12px | 12px | Light | 1.4 |

### Text Styling

**Headlines:**
```css
font-family: 'Poppins', sans-serif;
font-weight: 700;
letter-spacing: -0.02em;
```

**Body Text:**
```css
font-family: 'Poppins', sans-serif;
font-weight: 400;
line-height: 1.6;
```

**Labels & Tags:**
```css
font-family: 'Poppins', sans-serif;
font-weight: 600;
font-size: 12px;
letter-spacing: 0.1em;
text-transform: uppercase;
```

## Visual Elements

### Background Patterns

**Grid Pattern:**
```css
background-image: radial-gradient(
  circle at 1px 1px,
  rgba(255, 255, 255, 0.03) 1px,
  transparent 0
);
background-size: 40px 40px;
```

**Ambient Glows:**
```css
/* Top-left glow */
background: rgba(44, 138, 91, 0.05);
filter: blur(120px);
border-radius: 50%;

/* Used for ambient lighting effect */
```

### Card Styles

**Standard Card:**
```css
background: rgba(26, 58, 76, 0.3);
border: 1px solid rgba(255, 255, 255, 0.05);
border-radius: 16px;
```

**Spotlight Card (on hover):**
```css
/* Mouse-following radial gradient */
background: radial-gradient(
  600px circle at [mouse-x] [mouse-y],
  rgba(44, 138, 91, 0.15),
  transparent 40%
);
```

**Highlighted Card:**
```css
border: 2px solid #2C8A5B;
box-shadow: 0 0 30px rgba(44, 138, 91, 0.2);
```

### Buttons

**Primary Button:**
```css
background: #2C8A5B;

padding: 12px 24px;
border-radius: 8px;
font-weight: 500;
box-shadow: 0 4px 20px rgba(44, 138, 91, 0.2);
```

### Animations

**Entrance Animation:**
```css
/* Fade up */
from { opacity: 0; transform: translateY(20px); }
to { opacity: 1; transform: translateY(0); }
duration: 0.6s;
easing: ease-out;
```

**Hover Scale:**
```css
transform: scale(1.02);
transition: transform 0.2s ease;
```

**Gradient Animation:**
```css
@keyframes gradient-x {
  0%, 100% { background-position: 0% 50%; }
  50% { background-position: 100% 50%; }
}
animation: gradient-x 3s ease infinite;
```

## Voice & Tone

### Brand Voice Attributes
1. **Confident** - We know our methodology works
2. **Direct** - Clear, no jargon
3. **Empathetic** - We understand the stakes
4. **Technical but Accessible** - Expert knowledge, human delivery
5. **Honest** - Transparent about what we do and don't do

### Writing Guidelines

**Do:**
- Use active voice
- Be specific with numbers and claims
- Speak to the reader directly ("you", "your")
- Acknowledge challenges before presenting solutions
- Use British/SA English spelling (organisation, colour, analyse)

**Don't:**
- Use buzzwords without substance
- Overpromise or exaggerate
- Be condescending or overly technical
- Use excessive exclamation marks
- Make vague claims

### Key Phrases

**Value Propositions:**
- "Investment-grade financial models"
- "Mathematically proven"
- "Zero variance tolerance"
- "Computational validation"
- "60% faster, 75% lower cost"

**Trust Builders:**
- "Battle-tested in [region]"
- "$875M+ in funded projects"
- "100% audit pass rate"
- "Full audit trail included"

**Calls to Action:**
- "Get a Free Assessment"
- "Request a Quote"
- "See How It Works"
- "Download Methodology Guide"
