### Typography

```
PRIMARY FONT
────────────
Family: Century Gothic (or Poppins for web)
Weights: 400, 600, 700

SECONDARY FONT
──────────────
Family: Source Sans Pro (or Inter for web)
Weights: 400, 500, 600
```

### Icon Style

```
SPECIFICATIONS
──────────────
• Line weight: 2px stroke
• Corner radius: 2px
• Style: Modern, minimal, professional
• Colour: Monochrome (adapt to context)

ICON LIBRARY RECOMMENDATION
───────────────────────────
Primary: Lucide Icons (open source)
Alternative: Feather Icons
Custom: Only where necessary
```

## Image Guidelines

### Photography Style

```
GENERAL APPROACH
────────────────
• Professional, corporate aesthetic
• Clean, uncluttered compositions
• Natural lighting preferred
• Diverse representation
• Emerging market context

AVOID
─────
• Generic stock photo feel
• Overly staged scenarios
• Busy or distracting backgrounds
• Western-centric imagery
• Dated aesthetics
```

### Infographic Style

```
DESIGN PRINCIPLES
─────────────────
• Data-driven, not decorative
• Clear visual hierarchy
• Consistent with brand colours
• Professional, investment-grade
• Print-ready quality

FORMAT
──────
• SVG for scalability
• PNG fallback at 2x resolution
• Transparent backgrounds where applicable
```

---

## Technical Requirements

### File Formats

```
VECTOR GRAPHICS
───────────────
Format: SVG (preferred)
Fallback: PNG at 2x resolution
Use for: Icons, diagrams, illustrations

RASTER IMAGES
─────────────
Format: WebP (preferred) or PNG
Fallback: JPEG for photographs
Optimise: All images compressed for web

DOCUMENTS
─────────
Format: PDF for printables
Resolution: 300dpi minimum for print
```

### Responsive Considerations

```
BREAKPOINTS
───────────
Mobile:  < 640px
Tablet:  640px - 1024px
Desktop: > 1024px

IMAGE VARIANTS
──────────────
Provide 1x, 2x, and 3x versions for hero images
Use srcset for responsive loading
Mobile-specific crops where needed
```
