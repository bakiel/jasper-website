## Technical SEO Checklist

### On-Page
- [ ] Title tags include primary keyword
- [ ] H1 matches search intent
- [ ] Meta descriptions with CTA
- [ ] URL structure clean (/dfi/ifc not /page?id=123)
- [ ] Internal linking between related pages
- [ ] Image alt tags with keywords
- [ ] Schema markup for services

### Content
- [ ] 1,500+ words on pillar pages
- [ ] 800+ words on spoke pages
- [ ] FAQ section with schema
- [ ] Clear CTAs on every page
- [ ] Updated regularly (freshness signal)

### Technical
- [ ] Fast load time (<3s)
- [ ] Mobile responsive
- [ ] HTTPS
- [ ] XML sitemap
- [ ] Robots.txt configured
- [ ] No broken links
- [ ] Proper canonical tags
