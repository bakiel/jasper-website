## SEO Structure

### URL Strategy
- Clean URLs: `/services/growth` not `/services?type=growth`
- Trailing slashes: No
- Canonical tags: Self-referencing

### Page Titles
```
Home: JASPER Financial Architecture | DFI Financial Models
Methodology: JASPER Methodology | 10-Layer Financial Model Architecture
Services: Financial Modelling Services | JASPER
Growth: Growth Package | $12,000 | JASPER
[Sector]: [Sector] Financial Models | JASPER
```

### Meta Descriptions
- 150-160 characters
- Include primary keyword
- Include value proposition
- Include CTA element

### Heading Hierarchy
```
H1: One per page, includes primary keyword
H2: Section headings
H3: Subsection headings
H4: Rarely used
```

- Methodology diagrams (shows expertise)
- DFI logos (shows target market)
